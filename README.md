# Mini Moda

Sistema desenvolvido para o desafio prático da disciplina de Desenvolvimento Web.

## Integrantes

- Hortência
- Jéssica
- Riquelme

## Tema do Sistema

Mini Moda é um catálogo online de roupas e calçados infantis. O sistema permite visualizar os produtos disponíveis e solicitar um orçamento por meio de um formulário.

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- Supabase
- Git e GitHub
- GitHub Pages

## Explicação do Projeto

O projeto Mini Moda foi desenvolvido para facilitar a divulgação de roupas e calçados infantis. O site possui páginas de início, sobre, contato, solicitação de produtos e pedidos. Os usuários podem visualizar os produtos e realizar solicitações que ficam armazenadas em um banco de dados.

## Funcionalidades

- Página inicial com catálogo de produtos.
- Página "Sobre" com informações da empresa.
- Página de contato com informações da loja.
- Formulário para solicitação de produtos.
- Validação dos campos utilizando JavaScript.
- Armazenamento das solicitações no Supabase.
- Página para visualização dos pedidos cadastrados.

## Uso do Supabase

O Supabase foi utilizado como banco de dados do projeto. Quando o usuário envia uma solicitação na página "Solicitar Produto", as informações são gravadas automaticamente na tabela **pedidos** do Supabase. A página **Pedidos** consulta esse banco de dados e exibe todas as solicitações cadastradas.
