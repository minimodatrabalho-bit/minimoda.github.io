Sobre a Mini Moda

Conheça nossa história, nossos valores e o propósito deste projeto.

Quem Somos

A Mini Moda é uma loja virtual fictícia desenvolvida como projeto da disciplina de Desenvolvimento Web. Nosso sistema foi criado para oferecer uma experiência simples, moderna e organizada na solicitação de roupas e calçados infantis, demonstrando na prática os conhecimentos adquiridos durante o semestre.

Nossa História

O projeto foi desenvolvido com o objetivo de unir criatividade e tecnologia, criando um catálogo online responsivo, intuitivo e integrado ao Supabase para armazenamento dos pedidos realizados pelos usuários. Cada etapa foi planejada para proporcionar uma navegação agradável e facilitar o atendimento aos clientes.

Missão

Oferecer uma plataforma organizada e prática para apresentação de produtos infantis, proporcionando facilidade e qualidade no atendimento.

Visão

Demonstrar como a tecnologia pode contribuir para melhorar a experiência do cliente através de um sistema moderno e funcional.

Sobre o Projeto

O Mini Moda é um sistema web desenvolvido para demonstrar, na prática, os conhecimentos adquiridos na disciplina de Desenvolvimento Web. O projeto simula uma loja de roupas e calçados infantis, permitindo que os usuários conheçam a empresa, visualizem informações de contato, solicitem produtos por meio de um formulário e acompanhem os pedidos cadastrados. Todas as solicitações são armazenadas no Supabase, garantindo uma integração entre o front-end e o banco de dados. O sistema foi desenvolvido utilizando HTML, CSS, JavaScript, GitHub e GitHub Pages, com foco em uma interface responsiva, organizada e de fácil utilização.
